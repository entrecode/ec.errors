API Doc

newError

Helper Function to produce an error object containing message and HTTP status code.

newError(code: (int | string?), detail: string?, verbose: string?): Error
Parameters
code ((int | string?) = '000') internal error code without system identifier (three digits). Can also be an integer (when not 000…).
detail (string? = '') subject the error concerns.
verbose (string? = '') even more information about the error.
Returns
Error: An JavaScript Error instance.

convertValidationError

Converts an Error of tini validator into entrecode error format.

convertValidationError(tv4Result: object): Error
Parameters
tv4Result (object) Error from tini validator.
Returns
Error: Mapped error in entrecode format.

getLocalised

Create a localized version of any given entrecode error.

getLocalised(error: Error, locale: string?): Error
Parameters
error (Error) Error in entrecode error format.
locale (string? = config.locale) Two character localization (currently only 'de' supported). Leave blank for english.
Returns
Error: localized entrecode error.
